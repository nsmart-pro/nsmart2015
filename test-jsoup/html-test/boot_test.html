2.부트스트랩(Bootstrap) ?

-트위터의 UI디자이너 마크오토(Mark Otto) 와 개발자인 제이콥쏜튼(Jacob Thornton)이 개발

-HTML + CSS + 자바스크립트

-웹 UI 프레임워크, 웹 프런트 엔드 개발도구, 오픈형 UI 플러그인

-2011년 8년 오픈소스로 공개, 현재 깃허브(Github)에 수백개의 프로젝트가 존재

-웹페이지를 만드는데 필요한 버튼, 그리드, 폼과 같은 다양한 구성요소를 사용자에게 제공

- 2.x 버전은 데스크탑 환경에서의 처리가 중심, 3.x 에서는 모바일을 우선으로 지원
